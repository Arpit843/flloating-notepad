A floating notepad is a, often, lightweight application or digital tool that creates a small,
resizable, and, frequently, transparent window that remains "always on top" of other active programs.
It allows for real-time note-taking, multitasking, and keeping reminders visible without switching applications,
often with features like drag-and-drop, minimizing to icons, and automatic saving. 

Key features include:
Always-on-Top: Stays visible over other apps.
Multitasking: Allows taking notes while web browsing, watching videos, or gaming.
Customization: Often supports transparency, color changes, and resizing.
Functionality: Often includes checklist creation and drag-and-drop capability.
Convenience: Frequently saves data automatically to ensure no information is lost.
Versatility: Works on desktops as widgets or as mobile apps for quick,, sticky note-style, access
